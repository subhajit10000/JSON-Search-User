This project is a responsive user search web application developed using HTML, CSS, and JavaScript. It demonstrates how to fetch and display data from an external REST API using the Fetch API and asynchronous programming with async and await. The application retrieves user information from the DummyJSON Users Search API based on a search query entered by the user. By default, it loads information related to "John" when the page first opens, providing users with an immediate example of how the application works.

The interface is designed with a clean and modern layout that is both visually appealing and easy to use. A search bar is placed at the top of the page, allowing users to search for different people by entering a name and clicking the search button or pressing the Enter key. While the data is being fetched from the server, a loading message is displayed to inform the user that the request is in progress. If no matching users are found, the application displays a user-friendly message instead of leaving the page blank.

Each user's information is presented inside an attractive profile card. The cards include the user's profile image, full name, age, gender, email address, phone number, company name, and address. The responsive card layout automatically adjusts to different screen sizes, ensuring a smooth viewing experience on desktops, tablets, and mobile devices. Hover animations and subtle shadow effects enhance the visual appearance and improve user interaction.

The project also includes proper error handling to manage situations such as network failures or server errors. This prevents the application from crashing and provides feedback to the user if something goes wrong. Overall, this project is an excellent example of integrating APIs into a frontend application while practicing DOM manipulation, asynchronous JavaScript, responsive web design, and modern UI development techniques. It is suitable for beginners learning web development as well as intermediate developers who want to strengthen their understanding of API integration and dynamic content rendering in JavaScript.
